1. Classification with a single neuron¶

a) We start by defining the single neuron model (a single neuron in the hidden layer), using the following three functions:

  • The function initialize_parameters of the neuron. It randomly initializes the model's weights with small values. We Initialize the bias with 0. The number of weights required is equal to the length of the image vector.
  • A function sigmoid which computes the sigmoid activation function
  • A function neuron that given an input vector, the weights and bias, computes the output of the single neuron model
In [20]:
def sigmoid(z):
    return 1/(1+np.exp(-z))
In [21]:
def initialize_parameters(dim):
    w = np.random.rand(dim,1)/1000
    b = 0
    return w, b
In [22]:
def neuron(w,b,X):
    pred_y = np.matmul(w.T,X)+b
    pred_y=sigmoid(pred_y)
    return pred_y

b) Forward Pass: We use the three functions above to compute a first forward pass for the input matrix $X$ containing the loaded dataset, for some initialization of the weights and bias.

\begin{align} Y_{\rm pred}=\sigma(w^\top X+b) = [y_{\rm pred}^{(1)},y_{\rm pred}^{(2)},\dots,y_{\rm pred}^{(m)}] \end{align}

c) Cost estimation:

We will use a binary cross-entropy loss, so that the empirical risk can be computed as: \begin{align} E = - \frac{1}{m} \sum_{i=1}^m y^{(i)} \log(y_{\rm pred}^{(i)}) + (1-y^{(i)}) \log(1-y_{\rm pred}^{(i)}) \end{align}

The following cross-entropy function gives as result the scalar cost value computed over the entire dataset

In [27]:
def crossentropy(Y,Ypred):
    
    cost =  0
    m=len(Y)
    for i in range(m):
        cost = cost - (Y[0,i]*np.log(Ypred[0,i]) + (1 - Y[0,i])*np.log(1 - Ypred[0,i]))
    cost = cost/m
    
    return cost

d) Back propagation:

After initializing the parameters and doing a forward pass, we need to backpropagate the cost by computing the gradient with respect to the model parameters to later update the weights

The backpropagation function which receives as input the the training set (X,Y), as well as the current predictions and returns the gradients updates for the weights and bias

In [28]:
def backpropagate(X, Y, Ypred):
    m = X.shape[1]
    
    #find gradient (back propagation)
    dw=np.matmul(X,(Ypred - Y).T)/m
    db = np.sum(Ypred - Y)/m
    grads = {"dw": dw,
             "db": db} 
    
    return grads

e) Optimization After initializing the parameters, computing the cost function, and calculating gradients, we can now update the parameters using gradient descent.

We will use the functions implemented above to fill_in the "gradient_descent" function that optimizes the parameters given a training set X, Y, a fixed number of iterations, and a learning_rate.

At each iteration the function computes the predictions of the training set, then computes the cost and the gradient and updates the parameters using backpropagation.

The cost is stored at each iteration.

In [29]:
def gradient_descent(X, Y, iterations, learning_rate):
    costs = []
    w, b = initialize_parameters(train_x.shape[0])
    
    for i in range(iterations):
        Ypred = neuron(w,b,train_x)
        cost = crossentropy(Y,Ypred)
        grads= backpropagate(X, Y, Ypred)
        
        #update parameters
        w = w - learning_rate*grads["dw"]
        b = b - learning_rate*grads["db"]
        costs.append(cost)
        
        if i % 100 == 0:
            print ("Cost after iteration %i: %f" %(i, cost))
       
    return w,b, costs

w, b, costs = gradient_descent(train_x,train_y,iterations=2000, learning_rate = 0.005)

g) Early stopping

In order to avoid overfitting we will use early stopping method. To do this we will take a part of the training set for validation. At each iteration of the gradient descent function we will compute the cost for the training set and the validation set, use the first one to update the parameters and the second one to avoid overfitting ( i.e keep updating if the validation cost is still decreasing and stop if not)

We will also use the validation set to chose hyperparameters (learning_rate, iterations) that give the best results

We will start by adapting the gradient descet function to include early stopping

In [32]:
from sklearn.model_selection import train_test_split

def gradient_descent_early_stopping(X, Y, iterations, learning_rate, percentage):
    
    Ntrain = np.rint((1-percentage)*X.shape[1]).astype(int)
    Nvalid = X.shape[1]-Ntrain
    train_x = X[:,:Ntrain]
    val_x = X[:,Ntrain:]
    train_y = Y[:,:Ntrain]
    val_y = Y[:,Ntrain:]
    
    w, b = initialize_parameters(train_x.shape[0])
    

    train_costs=[]
    val_costs=[]
    w_best=w
    b_best=b
    k=0
    
    for i in range(iterations):
        
        
        Ypred_train = neuron(w,b,train_x)
        Ypred_val = neuron(w,b,val_x)
        train_cost = crossentropy(train_y,Ypred_train)
        #print(train_cost)
        train_costs.append(train_cost)
        val_cost = crossentropy(val_y,Ypred_val)
        #print(val_cost)
        val_costs.append(val_cost)
            
        grads= backpropagate(train_x, train_y, Ypred_train)
        
        #update parameters
        w = w - learning_rate*grads["dw"]
        b = b - learning_rate*grads["db"]
        
        if i>0:
            if val_cost < val_costs[-2]:
                w_best=w
                b_best=b
                k=i
    
    return k,w_best,b_best,train_costs,val_costs
